# Media Assets

Organized media structure for the website.

## Directory Structure

```
media/
├── logo/           # Website logo (40×40px display)
├── projects/       # Project images and videos
├── about/          # Profile images for About section
└── cv/             # CV/Resume PDF
```

## Usage

1. **Logo**: Add your logo to `media/logo/` and update the logo reference in your code
2. **Projects**: Add project media to `media/projects/`
3. **About**: Add profile images to `media/about/`
4. **CV**: Add your CV PDF to `media/cv/`

Each subfolder contains a README with specific instructions. Delete those READMEs once you've added your files.
